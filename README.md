## Folder Structure

The workspace contains two folders by default, where:

- `src`: The folder to maintain source files.
- `lib`: The folder to maintain dependencies.
    - `input`: This folder contains the .csv input Contract files for consumption by the program.
    - `output`: This folder contains an image of the programs output as well as the .csv file generated by the program.

## Dependency Management

The `JAVA PROJECTS` view allows you to manage your dependencies. The prettierrc.yaml file here is to configure the use of a Prettier 
plugin for the Java language to support formatting the .java file in a potentially more visually appealing style.
